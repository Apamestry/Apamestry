### 📂 Featured Technical Projects

#### 🛒 [FlaskKart-Microservices](https://github.com/Apamestry/FlaskKart-Microservices) | *Cloud & Microservices*
Deconstructed a traditional monolithic e-commerce system into an optimized, **9-service distributed architecture**.
*   **Key Architectures:** Implemented an API Gateway, centralized stateless JWT Authentication, and localized per-service isolated MySQL databases.
*   **Orchestration:** Containerized the ecosystem using **Docker** and streamlined multi-container deployments via **Docker Compose**.
*   *Monolith version available for architectural comparison [here](https://github.com/Apamestry/FlaskKart).*

#### ✈️ [SkyIQ: Flight Analytics Platform](https://github.com/brand7noronha/Flight-delay-prediction-project) | *Full-Stack & Data Modeling*
A predictive analytics application designed to forecast airline delays and optimize travel itineraries.
*   **Database Engineering:** Architected and normalized a complex **9-table relational database schema** in MySQL to handle high-frequency flight transactional data efficiently.
*   **Stack:** Python (Flask), MySQL, HTML5/CSS3.

#### 📦 [Medicine Inventory Tracker](https://github.com/Apamestry/Medicine-Inventory-Tracker) | *Advanced Data Structures*
A highly optimized internal management tool tailored for enterprise healthcare infrastructure.
*   **Algorithms:** Designed utilizing custom **Hash Maps** for $\mathcal{O}(1)$ rapid lookups alongside a **B-Tree** implementation to handle multi-index inventory search and range queries efficiently.
